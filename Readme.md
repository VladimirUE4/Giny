# Giny 

Giny est un émulateur pour Dofus 2.62 écrit en C#.

This repository contains :

* A authentification server
* A game server
* Some tools to edit game data
* A Worldmap editor
* A Database synchronizer
* A Protocol builter (Action Script 3 to C#)
* A Selfmade ORM
